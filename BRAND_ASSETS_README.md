# Brand Assets Setup Guide

The following brand asset files need to be placed in the `/public` directory:

## Root Level Brand Assets
- **logo.png** — Main SaukiMart logo (transparent bg, 480×120px)
- **logo_white.png** — White version for dark backgrounds
- **og_image.png** — Social share image (1200×630px)
- **favicon.ico** — Multi-size favicon (16/32/48)

## Icons Directory (`/public/icons/`)
All icon files should be placed in `/public/icons/`:

- **icon_16x16.png** — Favicon size
- **icon_32x32.png** — Favicon size
- **icon_48x48.png** — Favicon size
- **icon_64x64.png** — Shortcut icon size
- **icon_180x180.png** — Apple touch icon
- **icon_192x192.png** — PWA icon
- **icon_512x512.png** — PWA large icon
